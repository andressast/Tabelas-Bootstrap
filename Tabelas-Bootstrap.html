    Uma linguagem bastante antiga mas que não sai de moda é a C. Criada em 1972, por Dennis Ritchie, é uma linguagem de baixo nível, muito comum na programação de sistemas operacionais, mas que também pode ser utilizada na criação de softwares para desktop – desde que auxiliada pela utilização de uma linguagem mais amigável com a criação de interfaces, como a C++.
    Apesar de já ser considerada uma linguagem antiga por muita gente da comunidade de programadores, Java ainda é bastante comum e costuma ser utilizada em diversos tipos de aplicações atualmente. Justamente por ser versátil, essa linguagem ainda tem uma longa história pela frente, e aprender a programá-la vai possibilitar que você crie aplicações tanto para Web, quanto para Mobile e Desktop, sendo uma linguagem valiosa no mercado tecnológico.

    Criada em 1991 por Guido van Rossum, é uma linguagem bastante interessante, principalmente para quem está começando a entrar no mundo da programação, pois um dos objetivos de sua criação foi proporcionar a facilidade de escrita e de leitura dos códigos em Python.
